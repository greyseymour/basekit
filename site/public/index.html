/01 · install

One command. Any environment.

Skills follow the open SKILL.md spec — portable across Claude Code, Cursor, Codex, Gemini CLI, and Windsurf.

$ claude plugin install basekit
$ cursor mcp add basekit
# or paste into .cursor/mcp.json:
{ "mcpServers": { "basekit": { "command": "npx", "args": ["-y", "@basekit/mcp"] } } }
$ codex mcp add basekit npx -- -y @basekit/mcp
$ windsurf mcp add basekit
# paste into ~/.codeium/windsurf/mcp_config.json:
{ "mcpServers": { "basekit": { "command": "npx", "args": ["-y", "@basekit/mcp"] } } }
$ git clone https://github.com/basekit/basekit
$ cd basekit && ./install.sh
# or copy ./skills/* into your agent's skill directory
After install: ask your agent "deploy an ERC-20 on Base sepolia" and the right skill activates automatically.

/02 · skills

Five playbooks. Zero copy-paste.

Each skill is a markdown playbook your agent loads on demand. Average size: ~100 lines of instructions, 0 lines of magic.

  1. 01

    base-deploy-token

    Ship an ERC-20 from prompt to verified contract on Basescan. Foundry under the hood, Aerodrome liquidity routing optional.

    trigger "deploy a token on Base"
  2. 02

    base-analyze-wallet

    Profile any address: activity, holdings, counterparties, risk signals. Forensics-grade, with citations.

    trigger "analyze 0x… on Base"
  3. 03

    base-gas-optimize

    Audit a Solidity contract for gas waste. Prioritized diffs, risk-flagged, calldata-conscious for Base economics.

    trigger "reduce gas in this contract"
  4. 04

    base-airdrop

    Merkle drops, sybil filtering, gas-paid-by-claimer pattern. Includes the claim contract and the proof generator.

    trigger "airdrop tokens to a list"
  5. 05

    base-basescan-debug

    Paste a failed tx hash. Get the decoded call tree, the failing frame, the root cause, and the fix — Tenderly link included.

    trigger "why did this tx fail on base?"
/03 · mcp

Six tools your agent can call directly.

Skills give your agent playbooks. The BaseKit MCP server gives it arms.

get_wallet_profile

Activity, holdings, counterparties for an address.

get_transaction_trace

Decoded trace, revert reason, failing frame.

estimate_token_deploy

Preflight gas + ETH cost at current base fee.

check_token_approvals

List active approvals; surface dangerous ones.

resolve_basename

Basename ↔ address, both directions.

simulate_swap

Quote on Uniswap V3 or Aerodrome, read-only.
